Este repositorio (`Ferchozo/Ferchozo`) es a la vez mi **README de perfil** y el código fuente del portafolio, desplegado en GitHub Pages.

### Rutas del sitio

| Ruta | Contenido |
|------|-----------|
| `/` | Landing principal |
| `/servicios` | Portafolio de proyectos |
| `/politica-privacidad` | Política de privacidad |
| `/proyectos/` | Demos estáticas (cafetería, facturación, inventario, PWA…) |

### Desarrollo local

```bash
npm install
npm run dev
# http://localhost:3000
```

### Build

```bash
npm run build          # local, sin basePath
npm run build:pages    # GitHub Pages → salida en out/
bash scripts/deploy.sh # copia out/ a la raíz (modo legacy)
```

### Despliegue

Cada push a `master` ejecuta `.github/workflows/pages.yml`.  
En **Settings → Pages**, la fuente debe ser **GitHub Actions**.
